Installing streamlit-avatar
==================

Installing `streamlit-avatar` from the main channel can be achieved by:

```
conda install streamlit-avatar
```

It is possible to list all of the versions of `streamlit-avatar` available on your platform with `conda`:

```
conda search streamlit-avatar
```
